# Starting Kit – Sample Code Submission

This starting kit provides a minimal working example to help you understand how to submit predictions to the challenge platform.
The structure of this starting kit is as follow : 

```
starter_kit.zip
├── sample_code_submission
│   └── model.py
├── input_data
└── README.md
```

## Input_data
In this file you have all the needed data for the challenge

## Submission Format

Your submission must be a **ZIP file** with the following structure:

```
submission.zip
└── model.py          # Required: Your model implementation
```

The `model.py` file must contain a `Model` class that implements the required methods as shown.
